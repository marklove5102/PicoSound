# Place your WAV files here

WAV files must be:
- Sample rate: 16kHz
- Bit depth: 16-bit
- Channels: Mono

Example files:
- sound1.wav
- sound2.wav
- sound3.wav

Convert using Audacity:
1. Tracks → Resample → 16000 Hz
2. Tracks → Mix → Mix Stereo Down to Mono
3. File → Export as WAV (Signed 16-bit PCM)
